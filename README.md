AI Interview Preparation Bot

Overview

AI Interview Preparation Bot is a Flask-based web application that helps students prepare for technical interviews.

The application generates interview questions, evaluates answers using AI, calculates readiness scores, provides feedback, awards badges, and recommends topics for improvement.

Features

- Domain Selection (Python, SQL, Web Development, Data Science)
- Difficulty Levels (Easy, Intermediate, Advanced)
- AI Generated Interview Questions
- AI Answer Evaluation
- Interview Readiness Score
- Progress Bar Visualization
- Performance Badges
- Strengths and Weaknesses Analysis
- Recommended Topics
- Detailed AI Feedback

Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Groq API

How to Run

1. Clone the repository

git clone <repository-url>

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open browser

http://127.0.0.1:5000
